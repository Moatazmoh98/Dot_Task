# Assesment Task

This project is a technical task for Vodafone.

## Project Specifications

This project contains 3 parts:

- An STLC for a mobile application.

- selenium java with cucumber framework for Vodafone eshop.

- Api test automation project using the rest-assured technique.

---

## Project structure
I made the structure for this basic project match the extendability for highly scale projects.

- Main folder
  - Pages.
  - TestData.
- Test folder
  - Features.
  - Rest assured.
  - Step definition.
  - TestRunner.
  - Utilities.
